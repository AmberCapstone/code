# boards/

A single compilable program. You should open VS Code directly in the board folder (i.e. the folder containing `platformio.ini`).

## Compiling

Run `PlatformIO: Build` from the Command Palette or click the "Check Mark" icon in the Status Bar.

## Uploading

Run `PlatformIO: Upload` or click the "Right Arrow" icon.

## Running Tests

Run `PlatformIO: Test` or click the "Flask" icon.
